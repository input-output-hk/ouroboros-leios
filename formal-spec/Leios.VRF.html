Leios.VRF

This module defines the Verifiable Random Function (VRF) interface and its use in the Leios protocol. It specifies the types and operations for VRF evaluation, verification, and key management, as well as predicates for block and vote eligibility based on VRF outputs. The module also provides the Leios-specific VRF abstraction and utility functions for protocol slot leader selection.
