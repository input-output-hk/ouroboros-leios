# Analysis of UTxO lifetime on mainnet

An analysis of the lifetime of UTxOs on Cardano mainnet and the size of the active UTxO set.


## Artifacts

- [tx-history.sh](tx-history.sh): Shell script to generate `.tsv` files for analysis.
- [tx-history.sql](tx-history.sql): Shell script to generate `.tsv` files for analysis.
- [tx-lifetime.ipynb](tx-lifetime.ipynb): Jupyter R notebook to analysis data and make plots.
